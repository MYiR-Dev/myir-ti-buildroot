MYD-AM335X

Intro
=====
To be able to use MYD-AM335X board of MYIR with the images generated by
Buildroot, you have to prepare the SDCard.

How to build it
===============

  $ make myd_defconfig

Then you can edit the build options using

  $ make menuconfig

Compile all and build rootfs image:

  $ make

Result of the build
-------------------

After building, you should get a tree like this:

  output/images/
  ├── am335x-myd.dtb
  ├── am335x-evm.dtb
  ├── MLO
  ├── rootfs.ext2
  ├── sdcard.img
  ├── u-boot.img
  ├── uEnv.txt
  └── zImage
