4.11 USB Host

4.11 USB Host

  本例程演示通过相关命令来验证USB Host驱动的可行性，实现读写U盘的功能。

测试硬件环境：

  • 硬件调试环境搭建见部署开发环境中的硬件调试环境搭建部分。
  • U盘一个
板型 MYD-AM335X MYD-AM335X-Y MYD-AM335X-J
USB Host接口 J3或J4 J19 J27

测试软件环境：

  • Linux Kernel 4.1.18
  • mount, umount 应用程序
